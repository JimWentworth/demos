Prompt Pattern Flip Cards

Explore prompt engineering patterns by flipping each card. Use the tabs to focus on related strategies without being overwhelmed by the entire list.

Persona

Persona Pattern

Create a persona for the model to shape how it responds.

How it helps

Define tone, expertise, and constraints so the model selects responses that align with the fictional persona.

Audience

Audience Persona Pattern

Shape the response based on who will receive it.

How it helps

Clarify audience needs, knowledge level, and tone so the model tailors its message effectively.

Tailoring

Tail Generation Pattern

Request responses tuned to a specific context or scenario.

How it helps

Guide the model to reflect niche requirements, edge cases, or highly specific constraints.

Dialogue

Flipped Interaction Pattern

Invite the model to interview you for missing details.

How it helps

Creates a back-and-forth dynamic that surfaces requirements before drafting a final response.

Play

Game Play Pattern

Turn prompting practice into a repeatable game.

How it helps

Engages users through rules, scoring, or challenges that reinforce effective prompt habits.

Collaboration

Ask for Input Pattern

Encourage the model to request clarification or feedback.

How it helps

Keeps the conversation interactive so gaps can be filled before the model finalizes its answer.

Verification

Fact Check List Pattern

Ask the model to validate statements against explicit criteria.

How it helps

Encourages scrutiny by prompting for corroboration, citations, or uncertainty when evidence is lacking.

Control

Question Refinement Pattern

Iteratively tighten your question to guide the output.

How it helps

Provides scaffolding so users can specify format, depth, or boundaries before the model answers.

Template

Template Pattern

Use a structured prompt to produce consistent outputs.

How it helps

Applies pre-defined sections or phrasing that reduce variability while speeding up prompt crafting.

Vocabulary

Meta Language Creation Pattern

Invent shorthand or labels the model can use.

How it helps

Creates a custom lexicon so repeated concepts or workflows remain precise and concise.

Process

Recipe Pattern

Provide the model a step-by-step script to follow.

How it helps

Ensures ordered execution of tasks, especially when the response requires multiple stages.

Menu

Menu Actions Pattern

Offer the model a menu of next steps to choose from.

How it helps

Constrains generation to relevant actions and clarifies what should happen next.

Audit

Cognitive Verifier Pattern

Ask the model to disclose how it reached a conclusion.

How it helps

Surfaces the implicit evaluation criteria so you can gauge reliability and alignment.

Guidance

Few Shot Prompting

Prime the model with curated examples before it responds.

How it helps

Provides context that nudges the model toward the desired format, tone, or reasoning pattern.

Depth

Chain of Thought Prompting

Encourage step-by-step reasoning across connected questions.

How it helps

Produces transparent intermediate steps that improve complex or multi-part answers.

Focus

ReAct Prompting

Combine reasoning and action by responding to prior outputs.

How it helps

Loops the model through deliberate reasoning before deciding on the next operation or tool call.

Alternatives

Alternative Approaches Pattern

Prompt for multiple strategies or viewpoints.

How it helps

Broadens ideation by surfacing different angles, trade-offs, or solution paths.

Detail

Outline Expansion Pattern

Ask the model to elaborate on each part of an outline.

How it helps

Transforms bullet points into fleshed-out sections while preserving the original structure.

Filter

Semantic Filter Pattern

Define rules that screen out irrelevant or unsafe outputs.

How it helps

Sets semantic guardrails so the model focuses on responses that meet your criteria.
