Week 1: Opening the Case — What does it mean to "see" a pattern?

Learning Goals

  • Understand what constitutes a "pattern" in data and machine learning contexts
  • Recognize the difference between human pattern recognition and algorithmic pattern detection
  • Identify key questions to ask when beginning data analysis
  • Develop awareness of how prior assumptions influence pattern recognition

Activities for the Week

  • Introduction discussion: Share your initial understanding of patterns in data
  • Watch: "Pattern Recognition in Machine Learning" video lecture
  • Read: Chapter 1 - "What Machines See" from course text
  • Interactive exercise: Pattern identification in sample datasets
  • Participate in class forum: Discuss examples of patterns you've noticed in everyday data

Deliverable for Grading

Assignment: Pattern Observation Journal (Entry 1)

Submit a 300-500 word reflection that:

  • Describes one pattern you've observed in a dataset or real-world scenario
  • Explains what made this pattern noticeable to you
  • Discusses what assumptions you brought to your observation
  • Reflects on how a machine learning algorithm might "see" this same pattern differently

Due: End of Week 1

Points: 20
