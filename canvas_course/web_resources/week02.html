Week 2: The Shared Case File — Features, labels, targets

Learning Goals

  • Define and distinguish between features, labels, and targets in machine learning
  • Understand how data is structured for machine learning applications
  • Recognize the importance of feature selection in model performance
  • Learn basic terminology used in supervised learning contexts

Activities for the Week

  • Video lecture: "Understanding Features and Labels"
  • Read: Chapter 2 - "The Anatomy of Training Data"
  • Hands-on lab: Exploring the shared case file dataset
  • Group activity: Identifying features and labels in sample problems
  • Discussion: What makes a "good" feature?

Deliverable for Grading

Assignment: Feature Engineering Worksheet

Complete and submit a worksheet that:

  • Identifies features, labels, and targets in three provided datasets
  • Proposes potential new features that could improve model performance
  • Explains your reasoning for feature selections
  • Addresses potential challenges in feature extraction

Due: End of Week 2

Points: 25
