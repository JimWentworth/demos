LLM Model Families & Evolution

Track model families and their timeline. Search, filter, and compare.

    About this catalog

    This is a lightweight, client‑side site to help you track major LLM families (not every checkpoint) and see key milestones over time. Use the search and filters to narrow the view. Click cards to expand details.

    Tip: You can add or edit entries in data.js. The UI will update automatically when you refresh.

    Editing guidance
    • Add a new entry to window.__LLM_FAMILIES__ with fields that match existing items.
    • Use consistent wording for licenses: e.g., “Open‑weight”, “Closed / Hosted”, “Open”.
    • Modalities: pick from Text, Vision, Audio, Code, Embeddings.
    • For the timeline, append to window.__LLM_TIMELINE__ with phase, event, and why.
