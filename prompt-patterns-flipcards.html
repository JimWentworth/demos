Persona Pattern

Basic Pattern

Create a "persona" for the language model to help it select the appropriate response style and expertise.

What It Does

Instructs the AI to adopt a specific role, expertise level, or character to frame its responses appropriately.

Key Benefits

  • Tailors tone and complexity
  • Provides domain expertise
  • Creates consistent character
Example: "Act as a senior software architect with 15 years of experience. Explain microservices to me."

Basic Pattern

What It Does

Specifies the target audience so the AI can adjust complexity, terminology, and examples accordingly.

Key Benefits

  • Appropriate complexity level
  • Relevant examples and analogies
  • Suitable vocabulary
Example: "Explain quantum computing for a 10-year-old who loves video games."

Template Pattern

Basic Pattern

Use a pre-defined template to structure the prompt and generate a specific type of response.

What It Does

Provides a structured format with placeholders that the AI fills in to create consistent, formatted outputs.

  • Reusable structure
  • Easy to customize

Flipped Interaction Pattern

Interactive Pattern

Ask the GenAI to ask you questions to create a more comprehensive and tailored response.

What It Does

Reverses the typical interaction by having the AI ask clarifying questions before providing an answer.

Key Benefits

  • More precise outputs
  • Identifies missing context
  • Interactive refinement
Example: "I need help planning a trip. Ask me questions to understand my preferences before suggesting destinations."

Chain of Thought Prompting

Reasoning Pattern

Prompt the model with a series of related questions to generate a more comprehensive response.

What It Does

Encourages the AI to show its reasoning process step-by-step, leading to more accurate and explainable answers.

Key Benefits

  • Improved accuracy on complex tasks
  • Transparent reasoning
  • Better problem decomposition
Example: "Solve this step by step: If a train travels 120 miles in 2 hours, how far will it go in 5 hours?"

ReAct Prompting

Reasoning Pattern

Prompt the model with a response to its previous output to generate a more focused response.

What It Does

Combines reasoning traces with actions, allowing the AI to reason about what to do and then act on that reasoning iteratively.

Key Benefits

  • Dynamic problem-solving
  • Adaptive reasoning
  • Action-oriented outputs
Example: "Think about what information you need, then ask for it. Based on my answer, decide the next step."

Key Benefits
